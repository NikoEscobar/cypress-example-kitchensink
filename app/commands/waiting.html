Waiting

Examples of waiting for an amount of time or resource to resolve in Cypress, for a full reference of commands, go to docs.cypress.io

cy.wait()

To wait for a specific amount of time or resource to resolve, use the cy.wait() command.

cy.get('.wait-input1').type('Wait 1000ms after typing')
cy.wait(1000)
cy.get('.wait-input2').type('Wait 1000ms after typing')
cy.wait(1000)
cy.get('.wait-input3').type('Wait 1000ms after typing')
cy.wait(1000)

cy.server()

// Listen to GET to comments/1
cy.route('GET', 'comments/*').as('getComment')

// we have code that gets a comment when
// the button is clicked in scripts.js
cy.get('.network-btn').click()

// wait for GET comments/1
cy.wait('@getComment').its('status').should('eq', 200)
